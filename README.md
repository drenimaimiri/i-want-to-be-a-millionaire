# Kush Do të Bëhet Milioner

## Tech Stack

- **HTML5** - Page structure
- **CSS3** - Styling with animations, gradients, and responsive design
- **JavaScript (Vanilla)** - Game logic
- **Google Fonts** - Bebas Neue and Roboto
- **Canvas API** - Particle effects in the background

## How to Play

### Play Quiz
1. Click the "Luaj Kuizin" (Play Quiz) button
2. Answer questions with 4 options (A, B, C, D)
3. You have 30 seconds for each question
4. Use lifelines if needed:
   - **50:50** - Removes two wrong answers
   - **Ndihmë** (Hint) - Gives you a hint
   - **Thërre Miqun** (Call Friend) - Simulates a phone call with a friend
5. Reach the $1,000,000 prize!

### Create Your Quiz
1. Click "Krijo Kuizin Tënd" (Create Your Quiz)
2. Choose the number of questions (1-15)
3. Fill in the question, correct answer, and 3 wrong answers
4. Optionally add a custom hint
5. Click "Gjenero Linkun" (Generate Link) to get a shareable link

### Keyboard Controls
- **A, B, C, D** - Select corresponding answer

## Installation

Simply open `index.html` in your browser. No installation required.

```bash
# Or use a local server
npx serve .
# or
python -m http.server 8000
```

Then open http://localhost:8000 in your browser.
